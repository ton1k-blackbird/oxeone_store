# NOIR — Интернет-магазин одежды

Элегантный интернет-магазин с панелью администратора, построенный на Flask.

## 🚀 Запуск

```bash
# 1. Установить зависимости
pip install flask

# 2. Перейти в папку проекта
cd shop

# 3. Запустить сервер
python app.py
```

Сайт откроется на: http://localhost:5000

## 🔐 Вход в панель администратора

URL: http://localhost:5000/admin
- Логин: `admin`
- Пароль: `admin123`

## 📁 Структура проекта

```
shop/
├── app.py              # Основное приложение Flask
├── data.json           # База данных (создаётся автоматически)
├── requirements.txt
├── static/
│   ├── css/
│   │   ├── main.css    # Стили магазина
│   │   └── admin.css   # Стили админки
│   └── js/
│       └── main.js
└── templates/
    ├── base.html       # Базовый шаблон
    ├── index.html      # Главная страница
    ├── shop.html       # Каталог
    ├── product.html    # Страница товара
    ├── cart.html       # Корзина
    ├── checkout.html   # Оформление заказа
    └── admin/
        ├── base.html
        ├── login.html
        ├── dashboard.html
        ├── products.html
        ├── product_form.html
        └── orders.html
```

## ✨ Функционал

**Магазин:**
- Красивая главная страница с героем
- Каталог с фильтрацией по категориям
- Страница товара с добавлением в корзину
- Корзина и оформление заказа

**Панель администратора:**
- Дашборд со статистикой
- Добавление / редактирование / удаление товаров
- Просмотр заказов и изменение статусов
